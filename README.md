# 多功能应用平台

这是一个基于Nuxt 3构建的多功能应用平台，集成了多种工具和功能。

## 主要功能

- 搜索功能
- 工具页面
- 标签页面
- 内容预览

## 技术栈

- Nuxt 3
- Vue 3
- Tailwind CSS
- Supabase

## 开始使用

```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 构建项目
npm run build

# 预览模式
npm run preview
```
